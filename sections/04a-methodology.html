Methodology

Reward, State and Action Space

  • State Space:
    • Lidar readings (180-degree scan)
    • Relative position of the goal
  • Action Space:
    • Linear velocity (forward/backward)
    • Angular velocity (left/right turn)
  • Reward Structure:
    • Positive reward for reaching the goal
    • Negative reward for collisions

PPO vs DDPG Algorithm Selection

  • Proximal Policy Optimization (PPO):
    • Stable and efficient policy gradient method
    • Clipped objective to prevent large policy updates
  • Deep Deterministic Policy Gradient (DDPG):
    • Actor-Critic method for continuous action spaces
    • Uses experience replay and target networks

Domain Randomization for Sim2Real Transfer

  • Randomizing robot dynamics:
    • Mass variations
    • Friction coefficients
  • Sensor noise simulation:
    • Lidar noise models
    • Odometry drift
  • Environmental changes:
    • Distinguishable Environments
    • Different waypoints

CTDE - Centralized Task Allocation Decentralized Execution

  • Centralized Task Allocation:
    • Allocation of tasks to agents based on global information
    • Facilitates coordinated behavior among agents
  • Decentralized Execution:
    • Each agent acts based on local observations
    • Policies conditioned on individual agent's perspective
